# מטלה מתוקשבת בעברית — חסרי אונים או בעלי יוזמה?

קובץ זה כולל אתר מטלה מוכן להעלאה ל־GitHub Pages.

## הקבצים
- `index.html` — דף המטלה
- `style.css` — עיצוב
- `script.js` — שאלות, תשובות נכונות, ניקוד ובדיקה אוטומטית

## הוראות העלאה ל־GitHub
1. פתחי Repository חדש ב־GitHub.
2. העלי את שלושת הקבצים: `index.html`, `style.css`, `script.js`.
3. לחצי על Settings.
4. עברי אל Pages.
5. תחת Branch בחרי `main` ואז `/root`.
6. לחצי Save.
7. לאחר כמה דקות יופיע קישור לאתר.

## הערה חשובה
שאלות 6–11 נבדקות אוטומטית באופן מלא.
שאלות 1–5 נבדקות לפי מילות מפתח, לכן מומלץ שמורה תבדוק אותן לפני קביעת ציון סופי.
