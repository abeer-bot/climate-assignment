# מטלה מתוקשבת בעברית — עם איסוף תשובות ל־Google Sheets

האתר מחובר ל־Google Apps Script הבא:
https://script.google.com/macros/s/AKfycbxZosDaXXVMfxpHj__kOQ_BZDOYq8-aQJTtFg991zzRk8OoYjeuLf3s00z7aFqBitWVcQ/exec

## מה נאסף בטבלה
- תאריך ושעה
- שם תלמיד/ה
- כיתה
- שם מורה
- ציון
- אחוז
- כל תשובות התלמיד/ה לשאלות 1–11

## הוראות העלאה ל־GitHub Pages
1. פתחי את קובץ ה־ZIP וחלצי אותו.
2. העלי ל־GitHub את הקבצים שבתיקייה, לא את קובץ ה־ZIP עצמו:
   - `index.html`
   - `style.css`
   - `script.js`
   - `README.md`
3. היכנסי ל־Settings → Pages.
4. תחת Branch בחרי `main` ואז `/root`.
5. לחצי Save.
6. לאחר כמה דקות תקבלי קישור לאתר.

## בדיקה מומלצת לפני שליחה לתלמידים
1. פתחי את האתר.
2. מלאי שם תלמיד/ה לדוגמה, כיתה ושם מורה.
3. עני על כמה שאלות.
4. לחצי שליחה.
5. בדקי שהשורה נוספה לטבלת Google Sheets.

## הערה
שאלות 6–11 נבדקות אוטומטית באופן מלא.
שאלות 1–5 נבדקות לפי מילות מפתח, ולכן מומלץ שמורה תבדוק אותן לפני קביעת ציון סופי.
